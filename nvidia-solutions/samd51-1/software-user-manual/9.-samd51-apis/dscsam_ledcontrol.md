# DSCSAM\_LEDControl

This function controls the User LED specific to Jetson carrier.

```c
BYTE DSCSAM_LEDControl(BYTE Value);
```

{% tabs %}
{% tab title="Input Parameters" %}
| Name  | Description                          |
| ----- | ------------------------------------ |
| Value | <p>1 = enable,</p><p>0 = disable</p> |
{% endtab %}
{% endtabs %}

| Return Value     |
| ---------------- |
| Error code or 0. |

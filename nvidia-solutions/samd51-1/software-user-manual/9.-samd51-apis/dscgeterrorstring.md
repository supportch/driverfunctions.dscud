# DSCGetErrorString

Function to return the error string for a given error code

```c
char* DSCGetErrorString(BYTE error_code);
```

{% tabs %}
{% tab title="Input Parameters" %}
| Name        | Description                                                       |
| ----------- | ----------------------------------------------------------------- |
| error\_code | Error code for which error description string has to be returned. |
{% endtab %}
{% endtabs %}

| Return Value |
| ------------ |
| Error string |

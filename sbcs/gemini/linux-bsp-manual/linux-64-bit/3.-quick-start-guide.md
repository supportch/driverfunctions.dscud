# 3. QUICK START GUIDE

Connect the M.2 2242 in respective slot of Gemini board, the slot is shown in following picture and then power on the board. It directly boots to Ubuntu Desktop since automatic login feature is enabled in the OS.

![Figure 1:  M.2 2242 Slot in Gemini Board](broken-reference)

The Ubuntu-20.04.1 LTS OS has following user credentials as follows   &#x20;

| **User name** | **Password**                            | **User Type** |
| ------------- | --------------------------------------- | ------------- |
| dscguest      | dscguest                                | Administrator |
| diamond       | User can setup the password while login | Standard      |
| root          | welcome                                 | root          |

# Step 4: Serial Port Data Receiver Commands

Based on the serial connections enter the serial numbers like ser1 or ser2 or ser3 or ser4

**Command Format:**

_`# cat /dev/serX` - X is serial port number (Ex: 1,2,3,4)_

**Example:**

_`# cat /dev/ser1`_\

# APPENDIX C: AUDIO TESTING

In QNX 7.0, audio files can be played using the wave command.

**Note**: Headphone or Speakers should be connected in the Lineout port.

**Command Format:**

_`# wave file.wav`_

**Example:**

_`# wave file_example_WAV_1MG.wav`_\

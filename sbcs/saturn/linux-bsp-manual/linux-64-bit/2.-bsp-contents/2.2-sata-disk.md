The M.2 SATA disk includes a bootable runtime Linux Ubuntu-20.04.2 LTS OS image. * USB mouse and keyboard support
* USB 2.0 and 3.0 ports
* Serial ports RS-232/422/485
* M.2 SATA storage devices
